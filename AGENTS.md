# Project Design Guidelines
- Before creating any new pages, components, or UI elements, you MUST read and strictly adhere to `/design.md`.
- Maintain the Minimalist & Modern Tech-Forward style defined in the design document (Zinc/Black/White color palette).
- Apply the specific font families (Inter, Space Grotesk/Outfit, JetBrains Mono) and adhere to the spacing and border-radius rules.
- Ensure all new components use the predefined button presets, micro-interactions (framer-motion), and custom scrollbar styles to keep the project style highly unified.
